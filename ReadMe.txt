Aplicación web E-commerce de ArgenTech - empresa fantasía de tecnología.

El sitio cuenta con una galería de productos en el home, un menu superior 
para visualizar productos por categoria y tambien una vista de detalle 
ingresando a cada producto con la información del mismo.

Además, obviamente, posee carrito de compras al cual se puede acceder en 
todo momento desde un icono en la esquina superior derecha, en la barra de navegación.
El icono cuenta con un marcador que indica la cantidad de productos ya ingresados.

La pagina del carrito muestra un mensaje de carrito vacío en caso de que no tenga productos.
En caso de ya haber agregado alguno, muestra el detalle de la compra y un formulario para realizarla.

Una vez apretado el boton del formulario, la aplicación trabaja con la base de datos firebase y
genera la orden de compra. Acto seguido, se muestra una sweetAlert con el detalle de la misma (id, fecha, items, etc.)
